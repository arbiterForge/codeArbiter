This is the current contents of the [Feature Forge](/feature-forge/overview/): every codeArbiter
feature in **preview** right now. Preview features are opt-in and off by default. This list is
generated from the same allowlist that stamps the <span class="ca-badge" data-kind="preview">preview</span>
badge on the reference pages, so it stays in step with them automatically.

## Currently in Preview

  ))}
</ul>

When this list changes it is because a feature entered or left the Forge: a new preview was added,
or one was promoted to stable and is now on by default for everyone.
